彩蛋

最早的时候大家都会使用 Phantomjs 来实现headless，但是多多少少会有一些不足的地方。现在 Chrome 和 Firefox 都已经提供了相应的 headless 模式，而且 Splinter 也已经集成。

通过一个简单的参数即可启用 headless 模式，赶快去试试吧！（Ps: 我为这个功能做了一点微小的工作）
